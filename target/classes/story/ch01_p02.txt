[BRIEFING]
The power monitoring subsystem sends readings in bursts
of four. NODE 1 is already programmed to accumulate them
and output the total.

Write NODE 0 to read each value from INPUT and forward it
to NODE 1 via the RIGHT port ($F1). Values are 0 to 63.
